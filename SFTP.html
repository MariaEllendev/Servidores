O que é um Servidor?

Um servidor é um sistema de computador (hardware e software) que fornece recursos, dados, serviços ou programas a outros computadores, conhecidos como clientes, através de uma rede. Em essência, ele "serve" informações e funcionalidades para os clientes.

Servidor Web

Armazena e entrega páginas web. Hospedagem de sites e e-commerce.

Servidor de Arquivos

Armazena e gerencia arquivos de dados. Compartilhamento de documentos em rede.

Servidor de E-mail

Gerencia o envio e recebimento de e-mails. Comunicação corporativa.

Servidor de Banco de Dados

Armazena e gerencia dados estruturados. Aplicações com persistência de dados.



Protocolo SFTP (Secure File Transfer Protocol)

O SFTP (Secure File Transfer Protocol) é um protocolo que fornece recursos de transferência de arquivos, acesso e gerenciamento sobre qualquer fluxo de dados confiável. Geralmente é usado em conjunto com o SSH (Secure Shell).

Características do SFTP

  • Porta Padrão: 22 (padrão do SSH)
  • Autenticação: Usuário/Senha ou Chaves SSH
  • Segurança: É SEGURO - dados e credenciais criptografados via SSH
  • Base: Subsistema do protocolo SSH

Recomendação

Use SFTP para qualquer transferência de arquivos que envolva dados sensíveis ou credenciais. O SFTP é o padrão moderno e seguro para transferência de arquivos. O FTP deve ser evitado em ambientes de produção e redes não confiáveis.
